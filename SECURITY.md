# Security Policy

## Supported Versions

Security fixes are applied to the latest published version of this Elixir library.

## Reporting a Vulnerability

Please do not open public GitHub issues for security vulnerabilities.

Instead, report privately by emailing:

- cees9000@gmail.com

Include:

- A clear description of the issue
- Steps to reproduce
- Potential impact
- Any suggested remediation

You can expect an acknowledgment within 7 days.

After confirmation and a fix, a coordinated disclosure will be made.
